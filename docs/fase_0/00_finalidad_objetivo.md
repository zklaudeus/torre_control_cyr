# 00 — Finalidad y Objetivo

## Finalidad de la aplicación
La finalidad de esta aplicación es crear una herramienta web simple para Torre de Control que reduzca la dependencia del Excel manual. 
El proyecto busca optimizar el proceso actual que se realiza mediante hojas de cálculo, permitiendo cargar datos de forma más ordenada, calcular automáticamente el Resumen por Zona y generar un reporte Excel mejorado.

## Qué busca optimizar
- Eliminar la dependencia de fórmulas manuales en Excel que pueden romperse o desactualizarse.
- Reducir el tiempo de carga manual y la copia de datos entre distintas fuentes.
- Centralizar la información operacional del día en un solo sistema.

## Resultado esperado
Una beta funcional interna, no una plataforma completa. La aplicación permitirá a Torre de Control:
1. Seleccionar una fecha de análisis.
2. Crear o abrir el reporte diario.
3. Registrar o cargar brigadas del día.
4. Ingresar programación diaria por zona.
5. Calcular automáticamente el Resumen por Zona.
6. Revisar el resumen en pantalla.
7. Exportar un Excel mejorado.

## Regla de Oro
> Se carga lo mínimo.
> Se calcula lo máximo.
> Se exporta el reporte.
