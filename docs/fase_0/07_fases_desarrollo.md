# 07 — Fases de Desarrollo

La ruta de trabajo propuesta para la construcción de la aplicación Torre de Control CYR EISESA está dividida en las siguientes fases. El progreso debe ser escalonado, asegurando que cada fase funcione correctamente antes de avanzar a la siguiente.

- **Fase 0:** Documentación y preparación *(Fase actual)*
- **Fase 1:** Base técnica FastAPI + React
- **Fase 2:** Base de datos mínima
- **Fase 3:** Reporte diario
- **Fase 4:** Programación por zona
- **Fase 5:** Brigadas del día
- **Fase 6:** Parámetros básicos
- **Fase 7:** Resumen automático
- **Fase 8:** Integración datos reales Neon
- **Fase 9:** Exportación Excel
- **Fase 10:** Pulido beta funcional
