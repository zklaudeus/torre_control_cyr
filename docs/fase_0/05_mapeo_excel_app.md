# 05 — Mapeo Excel a App

El siguiente mapeo describe cómo las distintas hojas del Excel actual se transforman en módulos o características dentro de la nueva aplicación.

| Hoja de Excel Actual | Destino en la App (Módulo o Futuro) |
|----------------------|---------------------------------------|
| Parametros | Parámetros básicos CYR |
| Brigadas_Maestra | Catálogo o referencia futura |
| Control_Diario | Datos operacionales / vista futura |
| Resumen_Zonas | Pantalla calculada principal (Resumen por Zona) |
| Dashboard | Futuro dashboard |
| Ranking_Diario | Futuro ranking |
| Gráficos | Futuras visualizaciones |
| Bonificación | Fase futura |
| Instructivo | Documentación de negocio |

---

Este mapeo garantiza que ninguna información clave del proceso actual se pierda, y define claramente qué elementos forman parte de la beta funcional y cuáles se programarán en fases posteriores.
